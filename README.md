# NormanAndHouse
***
- The portrait of my colleague, Norman, is scanned by Structure Sensor + iteez3D + Sketchfab. 
- Download models from Sketchfab.
- Build in Unity with Google Cardboard SDK.
- View it with Cardboard + Android phone.
